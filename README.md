# Antigravity Token Visualizer

Antigravityとの会話によるトークン消費量をリアルタイムで可視化するVS Code拡張機能です。

## 機能
- **Live Tracking**: `.gemini` フォルダ内のログを監視し、会話のトークン数増加を自動的にグラフ化します。
- **Delta Display**: 直近の発言でどれくらいトークンが増えたか（+300など）を表示します。
- **Session Reset**: セッションのカウントをリセットするボタンを提供します。

## 使い方
1. アクティビティバーのアイコンをクリックしてサイドバーを開きます。
2. Antigravityと会話をすると、自動的にグラフが更新されます。
3. 計測をやり直したいときは「Reset」ボタンを押してください。
